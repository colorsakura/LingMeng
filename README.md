# 灵梦 (LingMeng)

灵梦是一款跨平台的大模型 AI 会话客户端，使用 Electron + React 构建。

## 技术栈

- **Electron** - 跨平台桌面应用框架
- **React 19** - UI 框架
- **TypeScript** - 类型安全
- **Vite** - 快速构建工具
- **TailwindCSS** - 样式框架
- **better-sqlite3** - 本地数据库
- **Zustand** - 状态管理

## 开发

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建
npm run build

# 打包发布
npm run dist
```

## 功能

- 多 AI 后端支持 (Kimi, DeepSeek 等)
- 会话历史本地存储
- Markdown 渲染
- 实时流式响应
